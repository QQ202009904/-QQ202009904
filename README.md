《华严经》云：
戒为无上菩提本，
应当具足持净戒，
若能坚持于净戒，
是则如来所赞叹。
学佛修行，戒是根本，没有戒的基础，学佛不可能成就。所以，不论在家、出家，一进佛门的第一件大事，就是受戒。因此，佛教的戒律，因为佛子身份的不同，分为在家戒和出家戒。其中在家戒分为三皈戒、五戒、八关斋戒和菩萨戒。菩萨戒者，乃诸佛之本源，行菩萨道之根本，也是在家菩萨自誓上求佛道、下化众生的最高乘戒。《菩萨戒经》云：受持菩萨戒有五种殊胜功德利益：
一者常得十方诸佛愍念守护；
二者临命终时，正见心欢喜；
三者所生之处，与诸菩萨为友；
四者功德成就、戒度成就；
五者今世后世，性戒福慧圆满。
